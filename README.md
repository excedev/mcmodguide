# Learn modding
This is a short guide about modding in Minecraft with learning resources.

# About

This list is made to help people learn Minecraft modding. It collects useful information, tools, and resources for different parts of making Minecraft mods, datapacks, and related projects.

Most of the resources here are free. If something costs money, it will be clearly marked. Paid resources are only included if they are well-known and actually helpful for learning modding.

You can suggest changes or improvements. Check CONTRIBUTING.md for how to contribute, and look at open issues for things that still need fixing or updating.

This guide focuses on Minecraft Java Edition modding, including:

Making mods with Forge, Fabric, or NeoForge
Datapacks and vanilla-style changes
Basic game logic and coding
Making textures, models, and animations
Setup tools like Java, Gradle, and IDEs

It does not try to list every tool or random download. Tools are only included if they are important for learning modding or getting started. Some asset tools are included when they help you make mods more easily (like modeling or texture tools).

This is not a step-by-step tutorial list. It is not meant to solve every small problem. Instead, it helps you understand how Minecraft modding works so you can solve problems on your own. It focuses on learning ideas like blocks, items, events, rendering, and data systems.

It also avoids content that is only for watching or entertainment unless it teaches something useful.

The goal of this guide is to help you go from copying tutorials to actually understanding how Minecraft modding works.

Disclaimer: This guide is not made or supported by Mojang or Microsoft. It is made for learning purposes and focuses mainly on Java-based modding.
